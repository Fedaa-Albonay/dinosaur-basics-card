# Dinosaur Info Card 🦖

A clean and well-documented information card project built to master the fundamentals of web development. This project focuses on structured HTML content and precise CSS styling.

## 🚀 Overview
This project serves as a practical application of **HTML5** and **CSS3** basics. It features a stylized card containing scientific information about dinosaurs, demonstrating a clear understanding of the web's building blocks.

## 🛠️ Technologies Used
- **HTML5**: Used for semantic structure (Headings, Paragraphs, Lists, and Anchors).
- **CSS3**: Used for layout design, color schemes, and the **Box Model** (margins, padding, and borders).

## 📚 Learning Resource
The core concepts applied in this project were learned from the **Codezilla** YouTube channel, which provided an excellent foundation for these web technologies.
- **Source:** [Codezilla - YouTube](https://www.youtube.com/@Codezilla)

## 💡 Key Features & Concepts Applied
- **Semantic HTML**: Using meaningful tags like `<hr>`, `<ul>`, and `<strong>` for better accessibility.
- **CSS Box Model**: Precise control over element spacing and border styling.
- **Link Handling**: Implementing external links with `target="_blank"` for a better user experience.
- **Code Documentation**: The source code is heavily commented to explain the purpose of each HTML element and CSS property, which is a great practice for future collaboration.

## 📁 Project Structure
- `index.html`: The core structure and content.
- `style.css`: The visual styling and layout rules.
- `/img`: Directory containing the project assets (Images).

## 💻 How to Run
1. Clone the repository or download the files.
2. Open `index.html` in any modern web browser.
